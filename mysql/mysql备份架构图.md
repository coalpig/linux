## 1.DBA备份与恢复职责

```
1.备份、恢复策略的设计。
  备份周期、备份工具、备份方式、恢复方式全部流程化
2.日常备份检查
  日志、备份内容  
3.定期的恢复演练
4.数据故障时，利用现有的资源，快速恢复
5.数据迁移、升级。
```
