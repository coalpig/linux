

## 1.命令触发

```
mysql> flush logs; 
shell# mysqladmin flush-logs
shell# mysql -e "flush logs"
shell# mysqldump -F
```

## 2.自动触发

```
mysql> select @@max_binlog_size;
+-------------------+
| @@max_binlog_size |
+-------------------+
|        1073741824 |
+-------------------+
```

注意：重启数据库会自动触发刷新日志

## 3.默认大小

默认1个G大小,超过了自动刷新

```
select @@max_binlog_size;
set max_binlog_size=字节;
```
