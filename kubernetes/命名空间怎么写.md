
需要定义一个资源类型为namespace也就是命名空间
```
apiVersion: v1
kind: Namespace
metadata:
  name: db
```
然后Deployment和svc和ingress可以绑定这个命名空间

```
apiVersion: apps/v1
kind: Deployment
metadata:
  name: wordpress-mysql
  namespace: db #绑定命名空间
  labels:
    app: wordpress-mysql
```

```
apiVersion: v1
kind: Service
metadata:
  name: wordpress-mysql-svc
  namespace: db #绑定
```
