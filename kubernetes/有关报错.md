```
Warning FailedCreatePodSandBox 5s kubelet, 10.0.0.23 Failed create pod sandbox: rpc error: code = Unknown desc = failed pulling image "docker.io/kubernetes/pause:latest": Error response from daemon: manifest for kubernetes/pause:latest not found: manifest unknown: manifest unknown
```

出现这种报错是因为kubelet配置pause官方镜像出问题

```
cat >/etc/kubernetes/cfg/kubelet<< EOF
KUBELET_OPTS="--logtostderr=true \
--v=4 \
--hostname-override=$(ifconfig eth0|awk 'NR==2{print $2}') \
--kubeconfig=/etc/kubernetes/cfg/kubelet.kubeconfig \
--bootstrap-kubeconfig=/etc/kubernetes/cfg/bootstrap.kubeconfig \
--config=/etc/kubernetes/cfg/kubelet.config \
--cert-dir=/etc/kubernetes/ssl \
--pod-infra-container-image=rancher/pause:3.6"
EOF
```

这样就可以把默认镜像改了
